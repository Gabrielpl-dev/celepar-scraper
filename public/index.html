Operações

Parâmetros da URL

Os demais campos vão como null/vazio igual à URL original.

Extrair Cod2 por cultura

script 01

Pesquisa todas as linhas cuja coluna "Cultura" bate exatamente com o nome informado (case e acento insensitivos). Retorna Cod2 e Alvo de cada uma.

Buscar por Cod2

script 02

Dado um Cod2, descobre em quais culturas ele aparece (deduplicando por nome de cultura).

Comparar duas culturas

script 03

Compara o conjunto de Cod2 entre duas culturas. Mostra exclusivos de cada uma e os comuns às duas.

Verificar produto

script 04

Procura linhas que contenham um termo (ex: nome de produto). Devolve o status pela cor do <font>: vermelho = tóxico · verde = seguro.

Listagem bruta

debug

Faz o GET na URL com os parâmetros do sidebar e devolve todas as linhas parseadas. Útil pra inspecionar.
